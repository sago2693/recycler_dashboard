# Gaiarec — Dashboard de la Asociación de Recicladores

Panel operativo para que la directiva de Gaiarec monitoree métricas de recolección, perfiles de recicladores y el directorio de productores de residuos.

Built with React + TypeScript + Vite, Tailwind CSS, shadcn/ui, and Recharts. Runs entirely in the browser with mock data — no backend required.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or later
- npm v9 or later

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The app will be available at **http://localhost:5173**.

## Other Commands

| Command | Description |
|---|---|
| `npm run build` | Build for production (outputs to `dist/`) |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint |
